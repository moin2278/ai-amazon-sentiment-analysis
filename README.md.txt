# AI Amazon Sentiment Analysis

## Overview
End-to-end NLP and machine learning project that predicts sentiment from Amazon product reviews.

## Dataset
Amazon Reviews (Musical Instruments category)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost, LightGBM
- TF-IDF
- Streamlit

## Models Trained
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM

## Features
- Text cleaning & preprocessing
- TF-IDF vectorization
- Model comparison
- Streamlit web app for predictions

## How to Run
```bash
pip install -r requirements.txt
streamlit run app/app.py
